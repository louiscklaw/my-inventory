### spin me up
'''
$ mkdir my-inventory
$ git clone git@github.com:louiscklaw/my-new-inventory.git
$ mkdir gh-pages
$ cd my-new-inventory
$ git checkout --orphan gh-pages
$ git worktree add ../gh-pages gh-pages
'''
